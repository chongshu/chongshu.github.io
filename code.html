[6/2019] Simulation Code for Shu, Chong 2019 "Endogenous Risk-Exposure and Systemic Instability".  | 

The code is written in Matlab. It is for the simulation of the network risk-taking distortion (risk-taking externality) proposed by Shu (2019) . Part of this code can also be used to solve the fixed point of a financial network's payment equilibirum, proposed by Eisenberg and Noe (2000). Code is available at Github
